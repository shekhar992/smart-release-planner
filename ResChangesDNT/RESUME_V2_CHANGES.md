## 💡 Pro Tips for Maximum Success

1. **Customize Profile for Each Job:**
   - Copy 5-7 keywords from job description
   - Weave them naturally into Profile paragraph
   - Example: If JD says "B2B SaaS" → add "B2B SaaS platforms" to Profile

2. **Add Metrics to Every Bullet:**
   - Always include: scale (users/devices), timeline (weeks/months), impact (% or $)
   - Use ranges when exact numbers aren't available ($XM, 500+, 15-person team)

3. **Optimize for FAANG/Tech:**
   - Emphasize: "Cross-functional leadership", "Data-driven", "Scale" (15,000 devices, 500+ users)
   - Add specific tech: GenAI/LLMs (RAG), React, TypeScript, AWS

4. **Print Test:**
   - Print the PDF (if sending to old-school companies)
   - Ensure it looks good in grayscale
   - No broken links or formatting issues
